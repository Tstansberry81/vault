# Coding Club

A Coding/AI club [[Traveler Stansberry]] helped run, for which he authored a **9-month curriculum** — and which is essentially the [[Homework Hatch (startup)|Homework Hatch]] skillset, taught.

## The curriculum
Monthly units building real software: coding/AI basics → Python & Cursor → **APIs (incl. pulling Canvas data)** → AI basics (OpenAI/Gemini API calls, a Q&A chatbot) → GitHub collaboration → **Flask web apps deployed on AWS Elastic Beanstalk** → AI for education (study-guide generators, ethics of AI in school) → hackathon/demo day → a capstone with LangChain/HuggingFace.

The "pull Canvas data and display it neatly" and "build a study-guide generator" projects are **direct precursors to his startup**. On his résumé his title is literally **"Co-Founder, Central Coder."**

## Tooling for the club (ChatGPT record)
A late-2025 chat shows him scoping a **free, easy, agentic coding setup** to deploy for the club — weighing **GitHub Copilot vs. [[Cursor (AI code editor)|Cursor]]** (his own daily driver) for students ([[GitHub Copilot vs Cursor - Free Agentic Coding Tools for the Coding Club (chat)]]).
