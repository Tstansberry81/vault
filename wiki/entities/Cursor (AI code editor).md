# Cursor (AI code editor)

The AI-assisted code editor [[Traveler Stansberry]] used as his primary development tool — the engine behind most of [[Homework Hatch (startup)]], which he built largely through **"vibe coding"** (prompting Cursor rather than writing code by hand).

## In his work
- He shipped the Homework Hatch codebase in Cursor, then wrote feature specs designed to feed straight into it ([[Cursor and the sisyphus10 Codebase Overview (chat)]]).
- A turning point: recognizing the limits of vibe coding, he set out to **actually learn to code** so he could understand and edit his own app ([[Learning to Code for Homework Hatch - Beyond Vibe Coding (chat)]]).
- He weighed Cursor vs. GitHub Copilot as the tool to teach in his [[Coding Club]].
